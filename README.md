# Atividade: Aplicação de WorkFlows com GitHub Actions

Este repositório é um exemplo preparado para a atividade prática de aplicação de WorkFlows com GitHub Actions.

Conteúdo:
- Exemplo simples em Node.js
- Testes com Jest
- Workflow GitHub Actions para CI (instalar dependências e rodar testes)

Como usar localmente:
1. Instale dependências: `npm install`
2. Rode os testes: `npm test`
3. Rode em dev: `npm start`

Arquivos importantes:
- `.github/workflows/ci.yml` — workflow de CI
- `src/` — código fonte
- `src/__tests__/` — testes
